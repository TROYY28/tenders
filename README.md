# TROY Procurement Directory

An App/Website that shows all tenders for your company's interest.

## Features
- National & State Owned Entities tenders
- Western Cape Municipalities tenders
- Direct links to 40+ government tender portals

## Preview
Open `tenders.html` in your browser or visit the live site at:
https://TROYY28.github.io/tenders/

## How to Use
1. Click on any tender link to view current opportunities
2. Links open in new tabs
3. All tenders are from official South African government sources
